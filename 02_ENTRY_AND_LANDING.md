# A. 전북현대 사이트 진입점

> 별도 데모용으로, 전북현대 데스크탑 사이트의 **헤더(유틸바+메인 네비)** 를 재현한 페이지를 하나 만든다.
> 이 페이지에서 PITCH OWNERS 메뉴를 클릭하면 우리 앱 랜딩(`/`)으로 이동한다.
> 목적: "전북현대 공식 사이트 안에 PITCH OWNERS가 자연스럽게 들어가 있다"는 흐름을 시연.

### 위치
`src/app/jeonbuk-demo/page.tsx` (데모 전용. 실제 운영 시 전북현대 측 사이트에 메뉴만 심으면 되므로, 이건 시연용 목업)

### 전북현대 헤더 재현 (데스크탑 풀폭)
레퍼런스 분석 기반:

**1) 상단 유틸바 (흰 배경, 높이 ~56px)**
- 좌측 보조 메뉴(텍스트, `text-[#333]`, 작게): `전북현대` · `그린스쿨` · `후원의집` · **`PITCH OWNERS`** · `오피셜 스토어` · `인권침해 접수처`
  - **`PITCH OWNERS` 항목을 신규 추가** — `후원의집` 바로 뒤에 배치. 후원 성격과 맥락이 맞음.
  - 이 항목만 시각적으로 살짝 강조: 딥그린 텍스트 + 우측 위 작은 `NEW` 또는 잔디 새싹 아이콘(lucide `Sprout`).
- 가운데: **JEONBUK / HYUNDAI MOTORS** 워드마크 로고 (딥그린 `#0B5D3B`, 영문 볼드 + 하단 작은 서브텍스트).
- 우측: `티켓예매` · `로그인` · `통합 회원가입`.

**2) 메인 네비게이션 바 (흰 배경, 하단 보더)**
- 좌측 햄버거 아이콘(`Menu`).
- 메뉴: `CLUB` · `TEAM` · `MATCH` · `TICKET` · `HISTORY` · `RECORD` · `MEDIA` · `FAN ZONE`
- 활성 탭(`CLUB`)은 **딥그린 배경 채움 + 흰 텍스트**(레퍼런스대로). 나머지는 `text-[#555]`.

**3) 히어로(간단 목업)**
- 더미 뉴스 히어로 한 장(다크 이미지 placeholder + "NEWS" 칩 + 제목 + `기사보기` 그린 버튼). 레퍼런스 톤만 흉내.

### 진입 동작
- 유틸바의 **`PITCH OWNERS`** 클릭 → `/`(우리 랜딩)로 이동(`<Link href="/">`).
- 추가로, `FAN ZONE`에 hover 시 드롭다운에도 `PITCH OWNERS — 내 잔디 입양하기` 항목 하나 더 노출(이중 진입 동선).

### 색상 메모 (이 데모 페이지 한정, 전북현대 화이트 테마)
- 배경 흰색, 텍스트 다크, 포인트 딥그린 `#0B5D3B`.
- ⚠️ 이 헤더 데모는 전북현대 화이트 테마를 따르되, **우리 앱 본체(`/` 이하)는 `00_FOUNDATION`의 다크 테마 유지.** 두 테마가 섞이지 않도록 이 페이지는 독립 스타일.

# B. PITCH OWNERS 전용 랜딩 페이지 (`/`)

> 다크 테마(`00_FOUNDATION` 토큰). 모바일 우선, max-w-480 컨테이너.
> 비전 → 문제 → 솔루션 → 입양 유도(CTA)의 스크롤 내러티브.
> 전북현대에서 넘어온 팬이 "이게 뭔지" 이해하고 격자 입양으로 전환되게 만드는 페이지.

### 섹션 구성 (스크롤 위→아래)

**S1. 히어로**
- 풀블리드 다크 배경 + 잔디 텍스처/그라운드 탑뷰 실루엣(SVG 또는 그라데이션 메시).
- 카피(메인): **"잔디 한 평을 가진 사람이 K리그의 진짜 주인이다."** (`font-display`, 큰 사이즈)
- 서브: "전북현대 그라운드의 잔디를 입양하고, 매 경기 당신의 격자에서 일어나는 모든 순간을 소유하세요."
- 1차 CTA(옐로우): **"내 격자 찾기"** → `/ground`
- 2차 CTA(고스트): "어떻게 작동하나요?" → 아래 S3로 스크롤.
- framer-motion: 로드 시 카피 스태거 등장.

**S2. 문제 제기 (왜 필요한가)**
- 짧은 임팩트 카피 + 숫자 강조. PRD 데이터 활용:
  - "서울월드컵경기장 잔디 관리비 2.5억 vs 대관 수익 82억 — **33배 격차**"
  - "EPL 한 구단 잔디 관리비 6.9~8.6억 vs 한국 1~2억"
  - "2025.9 한국-이라크전, 잔디 문제로 경기장 이전"
- 숫자는 `font-display` 옐로우로 카운트업 애니메이션(framer-motion / 간단 훅).

**S3. 솔루션 — 어떻게 작동하나 (3~4스텝)**
스텝 카드(세로 스택, 아이콘+제목+설명):
1. **격자 입양** — 442개 격자 중 내 자리를 고른다 (16㎡, 4인 쿼터제).
2. **NFC 라이센스 카드** — 실물 카드로 매치데이 인증.
3. **매치데이 퀘스트** — 내 격자에서 골/세이브 등 9종 데이터 이벤트 → 뱃지·리워드 자동 적립.
4. **Green Turf Fund** — 모든 재원은 잔디 관리에만 환원(투명 공개).
- 각 스텝 스크롤 진입 시 페이드/슬라이드.

**S4. 핵심 차별점 (가치 제안)**
- 카드 3~4개: "데이터-자산 연동" / "4인 쿼터제 독점 방지" / "재원 폐쇄 순환" / "내 잔디 서사".
- 팬 관점 한 줄 가치: "내 잔디에서 골이 났다는 평생의 서사."

**S5. Zone 미리보기**
- 그라운드 탑뷰 미니 SVG + Signature/Dynamic/Basic 3존 색 범례 + 가격대(6만~40만/반기).
- CTA: "그라운드 둘러보기" → `/ground`.

**S6. 신뢰 / 투명성**
- Green Turf Fund 환원 구조 간단 인포그래픽(B2C 90% / B2B 100% 환원, 외부 회계 감사).
- "격자 입양은 자산 소유가 아닌 후원·기부입니다" 법적 고지 문구.

**S7. 최종 CTA + 푸터**
- 큰 옐로우 CTA: **"지금 내 격자 입양하기"** → `/ground`.
- 푸터: 전북현대 로고 동반 표기("Official Partner / 전북현대모터스FC"), 약관·개인정보·문의 링크. (전북현대 사이트로 돌아가는 링크 = `/jeonbuk-demo`)

### 랜딩 전용 헤더
- 앱 하단 탭바 없음. 상단에 간단한 미니 헤더(PITCH OWNERS 워드마크 + "전북현대" 동반 + 로그인).
- 스크롤 시 1차 CTA가 하단에 sticky로 따라오는 옵션(모바일 전환율).

### Mock / 데이터
- 숫자·카피는 PRD 값 그대로 하드코딩 가능(랜딩은 정적 콘텐츠).
- 이미지·잔디 텍스처는 CSS 그라데이션/SVG로 대체(외부 에셋 의존 X).
